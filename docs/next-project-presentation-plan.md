# Next Plan: Project Presentation Upgrade

## Goal
Present each project as a concise case study with clear structure:
- What problem it solves
- Which stack was used and why
- How the solution flows end to end
- What outcomes and ownership were delivered

## Scope (Phase 1)
1. Extend project data model for case-study fields.
2. Populate current portfolio items with structured case-study data.
3. Redesign portfolio details page into sections:
   - Problem + approach
   - Stack by layer
   - Solution flow timeline
   - Impact, ownership, and tradeoffs
4. Keep existing filters and routing behavior unchanged.

## Scope (Phase 2)
1. Upgrade project cards with clearer outcome/problem snippets.
2. Add compact visual flow indicators directly on cards.
3. Add optional screenshots per flow step.
4. Add measurable impact metrics where available.

## Done Criteria
- Every project has a consistent case-study structure.
- Visitor can understand value, architecture, and execution flow in under 60 seconds.
- Page remains responsive on mobile and desktop.
